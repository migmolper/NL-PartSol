131  Get the jacobian of the transformation of the reference element :
132

137  - dNdX_Ref_GP -> Derivative gradient evaluated in the GP (2x4)
138 
139  Output :
140  - Jacobian -> Jacobian of the reference element
141  evaluated in the GP

149  Tensor F_Ref_I;

151 
152  /* 1º Evaluate the derivarive of the shape function in the GP */
153  dNdX_Ref_GP = Q4_dN_Ref(X_NC_GP);
154 
155  /* 2º Get the F_Ref doing a loop over the nodes of the element */
156  for(int I = 0 ; I<4 ; I++){
